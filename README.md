# Caesar-Cipher
![image](caesar-cipher.png)

##  🛠 Description
Simple Caesar Cipher encryptor and decryptor built with Python.

## ⚙️ Languages or Frameworks Used:
The script was created with [Python3](https://www.python.org/downloads) and the built-in functions in it.

## 🌟 How to run:
+ `python3 caesarcypher.py` 
+ `Enter string to decrypt: hello  -->  mjqqt`
